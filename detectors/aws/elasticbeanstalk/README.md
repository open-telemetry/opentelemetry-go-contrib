# AWS Elastic Beanstalk Resource Detector

[![Go Reference][goref-image]][goref-url]

This module provides a resource detector for AWS Elastic Beanstalk.

## Installation

```bash
go get -u go.opentelemetry.io/contrib/detectors/aws/elasticbeanstalk
```
